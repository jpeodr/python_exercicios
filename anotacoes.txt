* Tipos de dados: *

bool  - verdadeiro ou falso | ativo = true
int  -  inteiro | -1, 0, 1, 2...
float - flutuante | (2.5)
string - txt | "abacate"
list - lista | frutas = ["maça", "banana", "laranja"]
tuple - coordenada = (10.1, 20.4)
dict - dicionário = usuario = {"nome": "João", "email": "joao@exemplo.com"}

A função input() sempre obtém dados de entrada como texto (string 'str'), necessário conversão

 - Conversão de dados: 

int(dado_para_conversao): converte o dado para o tipo inteiro.
float(dado_para_conversao): converte o dado para o tipo float.
str(dado_para_conversao): converte o dado para o tipo string.
bool(dado_para_conversao): converte o dado para o tipo booleano.
